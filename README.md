# Ethereum Wallet
Simple react native ethereum wallet to show transactions.

# Instalation
install react native
https://reactnative.dev/docs/0.61/getting-started


create .env file
```javascript
INFURA_PROJECT_ID=
ETHERSCAN_API_KEY=
DEFAULT_WALLET=
NETWORK=
```

run it
```javascript
yarn && yarn android
```
